照片浏览器

本地优先的桌面照片 / 视频浏览器。打开文件夹即自动解析 EXIF、生成缩略图，按时间、相机、格式、定位智能分类，丝滑看图。文件不上传，隐私留在本机。

下载 macOS 版 GitHub

macOS · 需要 macOS 11+ · 视频功能需 ffmpeg

智能分类筛选

按类型、拍摄时间、相机品牌、格式、定位自动分组。点一下即看，缺失信息归「未知」也能筛。

原图清晰度

大图直读原文件呈现原始清晰度；HEIC/TIFF 等懒生成高分辨率预览，滚轮缩放以光标为锚，最高 8×。

照片 + 视频

递归扫描，照片识别 JPG/PNG/HEIC 等，视频识别 MP4/MOV/MKV 等。视频抽帧封面、可播放拖动进度。

隐私本地优先

所有处理都在本机完成，文件绝不上传任何服务器。索引、缩略图、日志按环境隔离存放。

万级不卡

虚拟滚动网格 + 自定义协议缩略图，扫描一次写入 SQLite，之后筛选排序全是数据库查询，丝滑流畅。

中英双语

界面与原生菜单中英双语，跟随系统语言自动切换，也可在「语言」菜单手动切换并记住。

安装说明

  1. 从上方下载 .dmg，拖入「应用程序」。
  2. 本应用未经 Apple 公证签名，首次打开请右键点击图标 → 打开，在弹窗里再点「打开」以绕过 Gatekeeper（之后双击即可）。
  3. 视频封面与元数据依赖 ffmpeg，可用 Homebrew 安装：brew install ffmpeg（未装不影响照片功能）。

系统要求：macOS 11 Big Sur 及以上。HEIC/HEIF 解码使用系统自带 sips。
